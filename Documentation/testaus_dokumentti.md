### Testaus

Testit voidaan suorittaa komennolla

```
mvn test
```

Testikattavuusraportti luodaan komennolla

```
mvn jacoco:report
```

## Checkstyle

Checkstylen voi suorittaa komennolla

```
mvn jxr:jxr checkstyle:checkstyle
```
